Basic Task Pipeline

  1. Gather all files from the source.

  2. Change or alter those files.

  3. Send those modified files to the destination.

Best Practices

1. Automate all Imports
  • inject
  • wiredep
  • useref
  • angular-file-sort

2. Establish directory structure
  • Source Code is not the same code served in browser
  • Production source code is minified and concatentated 
								
├── .tmp
│   ├── app.css
│   ├── app.js
│   ├── header.html
│   └── index.html
├── bower_components
│   └── angular
├── dist
│   ├── app.min.css
│   ├── app.min.js
│   └── index.html
└── src
    ├── app.scss
    ├── app.ts
    ├── components
    ├── header.jade
    ├── index.html
    └── shared

3. Production Build & Development Build
  • browser-sync
  • gulp serve vs. gulp serve:dist 
								
gulp.task('serve', [
  'inject:tmp',
  'watch-scss'
], function() {
  browserSyncInit([
    path.join(config.tmp, 'serve/'),
    './'
  ]);
});
gulp.task('serve:dist', [
  'build',
  'watch-scss'
], function() {
  browserSyncInit([
    config.dist,
    config.base
  ]);
});

5. Use gulp-useref for Production builds
  • Concatenation
  • Minification 
								
gulp.task('optimize',['inject', 'fonts'], function () {
 return gulp
  .src(config.tmp + 'serve/index.html')
  .pipe($.plumber ())
  .pipe($.useref())
  .pipe(jsFilter)
  .pipe($.uglify())
  .pipe(jsFilter.restore)
  .pipe(cssFilter)
  .pipe($.cleanCss())
  .pipe(cssFilter.restore)
  .pipe(gulp.dest(config.dist));
});

6. Separate tasks in separate files
  • require-dir 
								
//gulpfile.js
'use strict';
var gulp = require('gulp');
require('require-dir')('gulpTasks');
/**
 * Default task clean temporaries directories
 * and launch the main optimization
 * build task.
 */
gulp.task('default', [
  'clean'
], function() {
  gulp.start('build');
});
