What is Docker?

Docker is a platform that helps developers build, ship and run any application, in any environment. Docker containers wrap a piece of software in a complete filesystem that contains everything needed to run: code, runtime, system tools, system libraries, etc..

  • Docker Inc, Sponsor of Docker Platform
  • Offered as a Containers as a Service (CaaS)
  • 7 billion Docker Image Downloads
  • 3,000 Contributors
  • 700,000 Applications created using Docker

It is not a Virtual Machine

Virtual Machines (VM) are software computer, like a physical computer, runs and operating system and applications

  • Both provide isolated envrionment for applications to run inside.
  • Both are easily movable between differnet hosts.

The Trend

Monolithic

  • Cost a lot to run
  • Hard code base to debug and maintain
  • Unnecessary shared scope of responsibilites
  • Inflexible

Microservice

  • Smaller and Easier to run
  • Each service has a clearly defined set of responsibilities
  • Easy code base to maintain

    • Dilemmas with Microservice

    • Each service may have a different language or tech stack
    • Starting an application can be difficult due to the number of services needed to support the application
    • Each service may have its own dependencies and requirements

    Docker's Role

    Build,Run,Ship---Anywhere

    • Makes it easy for developers to write code that they can run anywhere by relyong on software containers
    • Containers are light weight and portable
    • Containers share the resources of the computer running them

    Pros & Cons

    Pro

    • Well Documented
    • Over thousands of public repositories
    • Fast

    Cons

    • Poor Monitoring
    • Platform dependent
    • Storage Dilemmas

    Run Your 1st Container

    Docker Image..are file system and configuration of our application which are used to create containers

    Going to run Alpine. Alpine is a lightweight Linux distribution.

    docker pull alpine

    To see that you pulled your alpine image run:

    docker images

    Lets run a docker container base off of this image

    docker run alpine ls -l

    Looked for the image. Created a container out of that image. Ran the command within it

    Try some linux commands like echo "Hello WWC"/ pwd / id -u -n/ cal

    For interactivity within your container run this:

    docker run -it alpine /bin/sh

    ls, uname -a, cd usr
