What is Front-End?

Front-End
Development

Also known as "client-side" development is the practice of producing HTML (Content), CSS (Presentation) and Javascript (Behavior) for a website or web application so that users can see and interact with them directly.

  • Cross Browser
  • Cross Device
  • Cross Platform

Cross Browser

Design for the web application to be viewable and accesible across all browsers.

  • Firefox
  • Chrome
  • Opera
  • Safari
  • Internet Explorer

Cross Device

Design for the web application to be viewed and accessible on all devices, regardless of:

  • Size
  • Device Browser
  • Brand

Cross Platform

Design for the web application to be viewed and accessible on all operating systems.

  • Linux
  • Mac OS X
  • Windows

      Cascade Styling Sheets

      Cascade Styling Sheets or CSS is the component in web development that focuses on the look and feel of the web application. That deals with the presentation layer of the application.

      To make cosmetic changes to the elements, you do so by using "Selectors". Selectors can style a single or multiple elements. 

      							
/* BLUEPRINT */
selector {
 property: value;
}
/* BLUEPRINT */

p {
 color: blue;
}
      To target a specific element on a page you can do that by using html attributes such as classes (.) and ids (#) . 
      							
.content-section {
 font-size: 30px;
}

#about {
 background-color: red;
}

      Responsive Web Design

      It is the approach that suggests that design and development should respond to the user's behavior and environment based on screen size, platform and orientation. The practice consists of a mix of flexible grids and layouts, images and an intelligent use of CSS media queries.

      • Grid System
      • Flexible
      • Media Queries

      CSS Frameworks

      A standardized package made up of a structure of files and folders of standardized code (HTML, CSS, JS documents etc.) which can be used to support the development of websites, as a basis to start building a site.

      • Bootstrap
      • Material Design
      • Foundation
      • Materialize

      CSS Preprocessor

      Is a tool that allows to bring CSS closer to using some basic concepts in programming. It incorporates selector nesting, functions for transformations, variables and mix-ins.

      • SASS
      • LESS
      • Stylus

      Document Object Model

      DOM for short, is is a standard for how to get, change, add, or delete HTML elements. It defines:

      • HTML elements as objects
      • Properties of all HTML elements
      • Methods to access all HTML elements
      • Events for all HTML elements

      Package Managers

      It is a piece of software that lets you manage the dependencies (external code written by you or someone else) that your project needs to work correctly.

      Module Loaders & Bundlers

      A bundler is a tool that puts your code and all its dependencies together in one JavaScript file and a module loader is is simply the process of stitching together a group of modules (and their dependencies) into a single file (or group of files) in the correct order.

      • Browserfiy
      • WebPack
      • RequireJS

      Frameworks

      An application which has defined open or unimplemented functions/objects, which the framework user writes to complete and customize his application. This usually takes the form of implementing an abstract function/interface, or inheriting from a known object.

      • AngularJS
      • VueJS
      • EmberJS
