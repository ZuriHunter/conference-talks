What is Front-End?

Front-End
Development

Also known as "client-side" development is the practice of producing HTML (Content), CSS (Presentation) and Javascript (Behavior) for a website or web application so that users can see and interact with them directly.

  • Cross Browser
  • Cross Device
  • Cross Platform

Cross Browser

Design for the web application to be viewable and accesible across all browsers.

  • Firefox
  • Chrome
  • Opera
  • Safari
  • Internet Explorer

Cross Device

Design for the web application to be viewed and accessible on all devices, regardless of:

  • Size
  • Device Browser
  • Brand

Cross Platform

Design for the web application to be viewed and accessible on all operating systems.

  • Linux
  • Mac OS X
  • Windows

      Cascade Styling Sheets

      Cascade Styling Sheets or CSS is the component in web development that focuses on the look and feel of the web application. That deals with the presentation layer of the application.

      To make cosmetic changes to the elements, you do so by using "Selectors". Selectors can style a single or multiple elements. 

      							
/* BLUEPRINT */
selector {
 property: value;
}
/* BLUEPRINT */

p {
 color: blue;
}
      To target a specific element on a page you can do that by using html attributes such as classes (.) and ids (#) . 
      							
.content-section {
 font-size: 30px;
}

#about {
 background-color: red;
}

      Responsive Web Design

      It is approach that suggests that design and development should respond to the user's behavior and environment based on screen size, platform and orientation. The practice consists of a mix of flexible grids and layouts, images and an intelligent use of CSS media queries.
